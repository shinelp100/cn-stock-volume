# 快速参考 - 用户操作清单

## 📋 生成报告 + 填充数据（5分钟快速方案）

### 第一步：生成报告框架
```bash
cd ~/.qclaw/workspace/skills/stock-daily-report
python3 scripts/generate_report.py 2026-03-27 --no-browser --output=~/Desktop/report.md
```

### 第二步：打开同花顺问财获取数据

#### 2.1 获取涨幅榜数据
1. 打开：https://www.iwencai.com/unifiedwap/result?w=近10日涨幅排名&querytype=stock
2. 等待页面加载
3. 记录以下信息：

| 数据项 | 位置 | 记录内容 |
|-------|------|---------|
| **主板冠军** | 表格中第一个 600/601/603 开头的股票 | 名称、代码、10日涨幅% |
| **创业板冠军** | 表格中第一个 300 开头的股票 | 名称、代码、10日涨幅% |
| **高位股票1** | 表格中涨幅最高的股票 | 名称、10日涨幅% |
| **高位股票2** | 表格中涨幅第二高的股票 | 名称、10日涨幅% |

**示例记录**：
```
主板冠军：华电辽能（600396）+131.70%
创业板冠军：海科新源（301292）+49.33%
高位股票1：普昂医疗 +136%
高位股票2：华电辽能 +131.70%
```

#### 2.2 获取涨跌家数比
1. 在同花顺问财搜索框输入：`涨跌家数`
2. 查看结果，记录：
   - 涨家数：XXXX
   - 跌家数：XXXX

**示例记录**：
```
涨家数：1500
跌家数：2500
比值：1500/2500 = 0.6 ≈ 1:1.7
```

### 第三步：编辑报告，填充数据

打开生成的报告文件：`~/Desktop/report.md`

找到以下部分并替换占位符：

#### 3.1 替换涨跌家数比

**查找**：
```markdown
   - 涨跌家数比：（数据获取中）（涨跌家数相对均衡）
```

**替换为**：
```markdown
   - 涨跌家数比：1500(涨) : 2500(跌) ≈ 1 : 1.7（下跌家数显著多于上涨，市场分化严重）
```

#### 3.2 替换市场反馈

**查找**：
```markdown
2. **市场反馈**：

  - 主板高度（10cm）：—（—）+0%，数据获取中
  - 创业板高度（20cm）：—（—）+0%，数据获取中
 - 高位股票：高位股数据获取中
```

**替换为**：
```markdown
2. **市场反馈**：

  - 主板高度（10cm）：华电辽能（600396）+131.70%，高位震荡继续新高，目前有30个交易日翻2倍规则，后期走势高位震荡盘整
  - 创业板高度（20cm）：海科新源（301292）+49.33%，锂电池板块加强
 - 高位股票：普昂医疗 10 日涨幅超 136% 领涨，华电辽能 +131.70% 紧随其后，高位股整体表现一般，高位进入滞涨的阶段
```

### 第四步：保存报告

编辑完成后保存文件。✅ 完成！

## 📊 数据填充模板

### 涨跌家数比模板

```
涨跌家数比：{涨家数}(涨) : {跌家数}(跌) ≈ 1 : {比值}（{情绪}）
```

**情绪判断规则**：
- 涨 > 跌 × 2：涨多跌少，市场情绪偏暖
- 涨 > 跌：涨跌互现，市场分歧存在
- 跌 > 涨 × 1.5：下跌家数显著多于上涨，市场分化严重
- 其他：涨跌家数相对均衡

## ❓ 常见问题

**Q：如果找不到主板/创业板冠军怎么办？**

A：表格中可能没有该板块的股票。此时：
- 主板：使用表格中第一个 600/601/603 开头的股票
- 创业板：使用表格中第一个 300 开头的股票
- 如果都没有，保留占位符 `—（—）+0%`

**Q：涨幅数字怎么填？**

A：直接从同花顺表格中复制，格式为 `+131.70%` 或 `-5.20%`

**Q：高位股票的"10日涨幅超"是什么意思？**

A：表示该股票在最近10个交易日内的累计涨幅超过该数值。例如 `+136%` 表示10日涨幅超过136%。

**Q：可以自动化吗？**

A：可以，但需要开发浏览器集成模块。详见 `BROWSER_AUTOMATION_GUIDE.md`

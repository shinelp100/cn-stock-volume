# 模板格式规范（强制执行）

## 概述

本报告生成器**严格按照固定模板格式输出**，不允许删减任何章节或字段。

## 强制规则

### 1. 章节完整性 ✅

报告**必须包含以下 6 个章节**，顺序不可更改：

| 章节 | 标题 | 说明 |
|------|------|------|
| 一 | 大盘指数解读 | 四市指数数据 + 成交量 + 操作策略 |
| 二 | 盘面理解与应对策略 | 市场情绪 + 市场反馈 + 总结 |
| 三 | 题材方向 | 至少 3 个题材（含行业逻辑 + 重点个股） |
| 四 | 明日计划 | 3 个观察题材 + 对应个股 |
| 五 | 近 10 个交易日涨幅前 20 股票 | 表格形式，必须 20 行 |
| 六 | 备注/其他 | 策略提示 + 数据来源 + 更新频率 |

**验证逻辑**：
```python
REQUIRED_SECTIONS = [
    "一、大盘指数解读",
    "二、盘面理解与应对策略",
    "三、题材方向",
    "四、明日计划",
    "五、近 10 个交易日涨幅前 20 股票",
    "六、备注/其他",
]
```

### 2. 数据完整性 ✅

**所有字段必须有值**，数据缺失时使用占位符：

- 指数数据缺失 → `"暂缺"`
- 股票数据缺失 → `"待补充"`
- 题材数据缺失 → 使用默认题材（人工智能、新能源、半导体）

**表格行数不足 20 只股票时**：
```python
# 自动填充占位符至 20 只
for i in range(len(gainers_list), 20):
    gainers_list.append({
        "股票代码": "待补充",
        "股票简称": "待补充",
        ...
    })
```

### 3. 格式一致性 ✅

**表格格式**：
```markdown
| 排名 | 股票代码 | 股票简称 | 收盘价 | 10 日涨幅 | 今日涨跌 | 人气热度 | 涉及概念 |
| :--: | :------: | :------: | :----: | :-------: | :------: | :------: | :------: |
| 1 | 688295 | 中复神鹰 | 60.59 | +99.97% | -0.05% | 100+ | 航空航天、新材料、碳纤维 |
...
| 20 | 待补充 | 待补充 | 待补充 | 待补充% | 待补充% | 待补充 | 待补充 |
```

**涨幅格式**：
- 必须包含 `+` 或 `-` 符号
- 保留 2 位小数（如 `+99.97%`）

**题材方向格式**：
```markdown
### 1. 题材名称（X 只股票）
- **行业逻辑**：行业逻辑描述
- **重点个股**：股票 1、股票 2、股票 3
```

### 4. 免责声明 ✅

**必须包含免责声明**，位置固定在报告末尾：

```markdown
⚠️ **免责声明**：本报告基于公开数据整理，仅供参考，不构成任何投资建议。股市有风险，投资需谨慎。
```

## 验证流程

### 自动验证（步骤 6 执行）

```python
def _validate_template_format(self) -> bool:
    """严格验证报告格式是否符合模板要求"""
    errors = []
    warnings = []
    
    # 1. 检查必需章节
    for section in REQUIRED_SECTIONS:
        if section not in self.report_content:
            errors.append(f"缺少必需章节：{section}")
    
    # 2. 检查必需字段
    for field in REQUIRED_FIELDS:
        if field not in self.report_content:
            warnings.append(f"缺少字段：{field}")
    
    # 3. 检查表格行数
    if len(table_lines) < 20:
        warnings.append(f"表格行数不足：{len(table_lines)}/20")
    
    # 4. 检查免责声明
    if "免责声明" not in self.report_content:
        errors.append("缺少免责声明")
    
    return len(errors) == 0
```

### 验证失败处理

| 问题类型 | 处理方式 |
|----------|----------|
| 章节缺失 | ❌ 错误，自动修复后重新生成 |
| 字段缺失 | ⚠️ 警告，使用占位符填充 |
| 表格行数不足 | ⚠️ 警告，自动填充至 20 行 |
| 免责声明缺失 | ❌ 错误，自动添加 |

## 修改模板

如需修改模板格式，请同时更新以下文件：

1. `templates/report_template.md` - 模板定义
2. `scripts/generate_report.py` - 生成逻辑
3. `TEMPLATE_RULES.md` - 格式规范（本文件）

**修改后必须运行测试**：
```bash
python3 scripts/generate_report.py --test
```

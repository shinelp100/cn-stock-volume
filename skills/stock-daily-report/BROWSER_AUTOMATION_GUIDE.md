# 浏览器自动化指南 - 获取盘面理解数据

## 概述

本指南说明如何通过浏览器自动化获取"二、盘面理解与应对策略"下的数据：
- 涨跌家数比
- 主板高度（10cm）冠军
- 创业板高度（20cm）冠军
- 高位股票

## ⚠️ 重要限制

**OpenClaw 浏览器受到 SSRF 策略的严格限制**，无法自动导航到外部网站（包括同花顺问财）。

这是安全策略的限制，无法绕过。

## 前置条件

- OpenClaw 浏览器工具可用
- 同花顺问财网站可访问：https://www.iwencai.com/

## 用户操作步骤

### 方案 A：手动获取 + 粘贴（推荐快速方案）✅ 可用

#### 步骤 1：打开同花顺问财

1. 在浏览器中打开：https://www.iwencai.com/unifiedwap/result?w=近10日涨幅排名&querytype=stock
2. 等待页面加载完成

#### 步骤 2：获取涨幅榜数据

页面会显示近10日涨幅排名表格，包含：
- 排名
- 股票代码
- 股票名称
- 现价
- 当日涨跌幅
- 区间涨幅

**需要记录的信息**：
- 第一个主板股票（600/601/603开头）：名称、代码、10日涨幅
- 第一个创业板股票（300开头）：名称、代码、10日涨幅
- 涨幅 > 50% 的前2-3只股票：名称、10日涨幅

#### 步骤 3：获取涨跌家数比

在同花顺问财查询框输入：`涨跌家数`

页面会显示：
- 涨家数：XXXX
- 跌家数：XXXX

**需要记录的信息**：
- 涨家数
- 跌家数

#### 步骤 4：填充数据到报告

编辑生成的报告文件，找到"二、盘面理解与应对策略"部分，替换占位符：

```markdown
2. **市场反馈**：

 - 主板高度（10cm）：{主板名称}（{主板代码}）{主板涨幅}%，高位震荡继续新高，目前有30个交易日翻2倍规则，后期走势高位震荡盘整
 - 创业板高度（20cm）：{创业板名称}（{创业板代码}）{创业板涨幅}%，锂电池板块加强
 - 高位股票：{高位1名称} 10 日涨幅超 {高位1涨幅}% 领涨，{高位2名称} {高位2涨幅}% 紧随其后，高位股整体表现一般，高位进入滞涨的阶段
```

以及涨跌家数比：

```markdown
   - 涨跌家数比：{涨家数}(涨) : {跌家数}(跌) ≈ 1 : {比值}（{情绪描述}）
```

**示例**：
```markdown
2. **市场反馈**：

 - 主板高度（10cm）：华电辽能（600396）+131.70%，高位震荡继续新高，目前有30个交易日翻2倍规则，后期走势高位震荡盘整
 - 创业板高度（20cm）：海科新源（301292）+49.33%，锂电池板块加强
 - 高位股票：普昂医疗 10 日涨幅超 136% 领涨，华电辽能 +131.70% 紧随其后，高位股整体表现一般，高位进入滞涨的阶段
```

## 常见问题

### Q1：为什么不能自动化？

**A**：OpenClaw 浏览器受到 SSRF（Server-Side Request Forgery）策略的严格限制，无法导航到外部网站。这是安全策略的设计，用于防止恶意请求。

### Q2：有其他方法吗？

**A**：可以考虑以下方案：
1. **使用用户浏览器**：用户在自己的浏览器中打开同花顺问财，手动复制数据
2. **API 集成**：如果同花顺提供 API，可以直接调用（但通常需要认证）
3. **爬虫方案**：使用专门的爬虫库（但需要遵守网站的 robots.txt）

### Q3：快速模式需要多长时间？

**A**：
- 生成报告框架：15 秒
- 打开同花顺获取数据：2 分钟
- 编辑报告填充数据：2 分钟
- **总计：~5 分钟**

### Q4：数据准确性如何保证？

**A**：
- 手动模式：用户直接从同花顺查看，准确性最高
- 数据来自官方数据源（同花顺问财）

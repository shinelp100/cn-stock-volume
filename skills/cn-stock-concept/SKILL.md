# 个股题材查询 Skill

## 功能

获取指定股票的题材概念信息，包括：
- 同花顺题材分类（概念板块）
- 题材热度/热度排名
- 题材涨停次数
- 相关题材的市场表现

## 使用场景

用户询问以下类型问题时触发：
- "XXX 股票有什么题材"
- "XXX 涉及什么概念"
- "XXX 的同花顺题材"
- "XXX 题材热度"
- "查询 XXX 的概念板块"
- "XXX 属于什么行业题材"

## 工作流程

### 步骤 1：解析用户需求

从用户问题中提取股票代码或股票名称：
- 如果用户提供了股票代码（如 600519），直接使用
- 如果用户提供了股票名称（如 贵州茅台），需要先获取股票代码

### 步骤 2：获取股票代码

如果用户只提供了股票名称，使用同花顺问财查询股票代码：

**方法：使用 JavaScript 导航到问财页面**
```javascript
window.location.href = 'https://www.iwencai.com/unifiedwap/result?w={股票名称}&querytype=stock';
```

### 步骤 3：访问题材页面

**个股题材页面 URL 格式：**
```
https://stockpage.10jqka.com.cn/{股票代码}/concepthot.html
```

例如：
- 贵州茅台：https://stockpage.10jqka.com.cn/600519/concepthot.html
- 比亚迪：https://stockpage.10jqka.com.cn/002594/concepthot.html

**备选方案：如果上述页面无数据，使用问财查询：**
```
https://www.iwencai.com/unifiedwap/result?w={股票代码}%20题材&querytype=stock
```

### 步骤 4：获取页面数据

使用 `browser` 工具获取页面内容：
1. 启动浏览器
2. 使用 JavaScript 导航到目标页面
3. 获取页面快照或截图

### 步骤 5：解析并输出数据

从页面中提取题材信息，格式化输出。

## 输出格式

```markdown
## 📊 {股票名称}({股票代码}) 题材分析

### 题材板块
| 题材名称 | 热度排名 | 今日涨停 | 近期涨停 | 相关股票数 |
|:---:|:---:|:---:|:---:|:---:|
| 白酒概念 | 12 | 0 | 2 | 35 |
| 超级品牌 | 25 | 0 | 1 | 28 |
| ... | ... | ... | ... | ... |

### 题材热度解读
- 该股票共涉及 X 个题材板块
- 热度最高的题材是：{题材名}
- 近期有涨停记录的题材：{题材名}

---

**数据来源：** 同花顺 (10jqka.com.cn)  
**查询时间：** YYYY-MM-DD HH:MM GMT+8
```

## 示例对话

**用户**: 查询贵州茅台的题材

**AI 执行步骤：**
1. 确定股票代码：600519（贵州茅台）
2. 导航到题材页面：https://stockpage.10jqka.com.cn/600519/concepthot.html
3. 获取页面快照
4. 解析题材数据
5. 输出格式化结果

## 注意事项

1. **SSRF 策略**：使用 JavaScript 导航方式避免被 SSRF 策略阻止
2. **页面加载**：确保页面完全加载后再获取数据
3. **备选方案**：如果 concepthot.html 无数据，尝试使用问财查询
4. **北交所股票**：北交所股票代码以 8 或 4 开头 URL 格式可能不同
5. **热度数据**：热度数据反映该题材在同花顺平台的热度，供短线参考

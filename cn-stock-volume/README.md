# cn-stock-volume - A 股四市成交数据查询

获取中国股市四市（沪市、深市、创业板、北交所）指定日期的成交金额、增缩量及比例，并输出四市合计总结。支持查询三市（沪市/深市/北交所）上涨/下跌家数。

## 功能特性

- 📊 **四市成交数据** - 沪市、深市、创业板、北交所
- 📈 **涨跌家数** - 上涨/下跌股票数量统计
- 📉 **增缩量分析** - 与前一交易日对比
- 🔄 **自动适配** - 非交易日自动使用最近交易日数据

## 使用方式

```bash
python3 ~/.jvs/.openclaw/workspace/skills/stock-data-monorepo/cn-stock-volume/scripts/fetch_volume.py [日期]
```

### 示例

```bash
# 查询最近交易日
python3 scripts/fetch_volume.py

# 查询指定日期
python3 scripts/fetch_volume.py 2026-03-21
```

## 数据源

- **首选**：东方财富网网页版（Browser 方案）
- **备用**：东方财富/新浪/腾讯 API

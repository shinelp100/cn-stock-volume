# cn-stock-volume v1.2.2 更新说明

## 发布日期
2026-03-21

## 新增功能

### 📈 三市涨跌家数统计

新增查询三市（沪市/深市/北交所）的上涨/下跌/平盘家数功能，帮助用户快速了解市场情绪。

#### 数据源
- 使用东方财富指数 API
- 字段说明：
  - `f113`: 上涨家数
  - `f114`: 下跌家数
  - `f115`: 平盘家数

#### 输出内容
1. **上涨家数** - 三市上涨股票总数及占比
2. **下跌家数** - 三市下跌股票总数及占比
3. **平盘家数** - 三市平盘股票总数
4. **总计** - 三市股票总数
5. **市场情绪判断** - 根据上涨比例自动判断：
   - 🟢 强势：上涨比例 > 60%
   - 🟡 偏强：上涨比例 50%-60%
   - ⚪ 震荡：上涨比例 40%-60%
   - 🟠 偏弱：下跌比例 50%-60%
   - 🔴 弱势：下跌比例 > 60%

## 技术实现

### API 接口
```
https://push2.eastmoney.com/api/qt/stock/get
参数：
- secid: 1.000001 (上证指数), 0.399001 (深证成指), 0.899050 (北证 50)
- fields: f113,f114,f115 (涨跌家数字段)
```

### 代码变更
- 新增 `query_market_advance_decline()` 函数
- 新增 `build_advance_decline_summary()` 函数
- 修改 `print_report()` 函数，添加涨跌家数输出
- 修改 `main()` 函数，调用涨跌家数查询

## 使用示例

```bash
# 查询今日（包含涨跌家数）
python3 scripts/fetch_volume.py

# 查询指定日期
python3 scripts/fetch_volume.py 2026-03-20

# JSON 输出（包含涨跌家数详细数据）
python3 scripts/fetch_volume.py 2026-03-20 --json
```

## 注意事项

1. **非交易日处理**：周末/节假日查询时，涨跌家数数据可能为最近交易日数据
2. **数据延迟**：盘中数据实时变化，收盘后数据固定
3. **市场覆盖**：目前支持沪市、深市、北交所，创业板涨跌家数包含在深市中

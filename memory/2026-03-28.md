## 完成任务：stock-daily-report skill v3 精修版本

### 背景
用户要求更新 A 股每日复盘报告生成器，严格按照新的模板格式输出，包括六个章节：
1. 大盘指数解读
2. 盘面理解与应对策略
3. 题材方向
4. 近10日涨幅前20股票
5. 明日计划
6. 备注/其他

### 核心改进

#### 1. 数据源架构重构
- **四大指数**：腾讯财经 API（实时）+ 东方财富 K线 API（历史日期）
- **三市成交量**：东方财富 K线 API（沪市/深市/北交所）
- **市场情绪**：AkShare 全市场行情（可选，`--no-browser` 跳过）
- **题材方向**：东方财富概念热度 API + 腾讯实时股价
- **涨幅榜**：同花顺问财浏览器抓取（需手动注入）

#### 2. 脚本模块化
```
scripts/
├── fetch_volume.py       # 三市成交量（复用 cn-stock-volume 逻辑）
├── fetch_sentiment.py    # 市场情绪（AkShare）
├── fetch_concepts.py     # 题材方向 + 实时股价
├── fetch_top10.py        # 涨幅榜解析（浏览器注入）
└── generate_report.py    # 主脚本（数据聚合 + 模板渲染）
```

#### 3. 模板格式精修
- 六章节结构完整
- 缩进规范：模板 3 空格 + 内容补充空格 = 目标总缩进
- 支持历史日期查询（自动切换 K线 API）
- 支持 `--no-browser` 快速模式（跳过 AkShare + 涨幅榜）
- 支持 `--output=path` 文件输出
- 支持 `--json` JSON 输出

#### 4. 关键数据字段
- **指数**：点位、涨跌幅、支撑/压力位
- **成交量**：三市合计、增缩额、增缩比例
- **情绪**：涨跌家数比、涨停/跌停数
- **题材**：行业逻辑、龙头板块、重点个股
- **涨幅榜**：排名、代码、名称、收盘价、10日涨幅、今日涨跌、人气热度、涉及概念
- **明日计划**：观察题材 × 对应个股
